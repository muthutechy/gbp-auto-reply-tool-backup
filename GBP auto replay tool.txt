You are a senior full-stack architect and SaaS engineer.

Build a complete production-ready AI-powered SaaS platform called:

"GBP SEO Auto Reply System"

This system is designed for a digital marketing agency to manage multiple Google Business Profile clients and automatically generate SEO-optimized replies to reviews using AI.

--------------------------------------------------

🎯 CORE REQUIREMENTS:

1. Multi-tenant SaaS architecture
   - Each client has a separate login
   - Data isolation per tenant
   - Agency admin can manage all clients

2. Google Business Profile Integration
   - OAuth 2.0 authentication
   - Store access_token and refresh_token securely
   - Fetch reviews using API
   - Post replies via API

3. AI Reply System (OpenAI API)
   - Generate human-like replies
   - SEO-optimized (natural keyword usage, NOT spam)
   - Include:
       - business name
       - primary keyword
       - optional location
   - Avoid repetition and keyword stuffing

4. Auto + Semi-Auto Workflow
   - Rating >= 4 → auto reply
   - Rating == 3 → auto reply
   - Rating <= 2 → send for client approval

5. Client Dashboard
   - Login system (JWT-based)
   - View all reviews
   - View AI-generated replies
   - Approve / edit / reject negative review replies

6. SEO Optimization Engine
   - Each client defines:
       - primary keyword
       - secondary keywords
       - location
       - tone (friendly / professional / formal)
   - AI must inject keywords naturally
   - Use semantic variations (not repetition)

7. Background Processing
   - Cron job every 5 minutes
   - Fetch new reviews
   - Queue system (BullMQ or similar)
   - Worker processes AI replies

8. Human-like Behavior
   - Random reply delay (2–15 minutes)
   - Reply variation (no duplicate templates)

9. Safety & Policy Engine
   - Prevent:
       - spammy replies
       - repeated keywords
       - promotional tone
       - links
   - Limit reply length (< 300 characters)

10. Analytics Dashboard
   - Per client:
       - total reviews
       - reply rate
       - avg response time
       - sentiment breakdown
       - keyword usage tracking
